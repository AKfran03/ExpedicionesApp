Expliaciones varias:

Estructurado de carpetas:
	-dao: van todas las clases que se comunicarán (select's,delete's,update's,insert's,etc.)
	-model: van las clases que le dan sentido al proyecto.
	-ui: Clases que hacen uso de Swing o demás interfaces gráficas.
	-útil: Clases que aplican utilidades, tales como conexión inicial a la base de datos, etc.


	-csv's: acá se encontrarán todos los archivos csv necesarios para importar a la base de datos.
	
	-Python: acá hay una serie de archivos.py y .csv, son todas las cosas que use para realizar la conversión al formato que necesitamos.
		si necesitan cambiar alguna columna o algo, desde los csv. son una buena base para mandarle a chat.
